# Discovery of Patterns in U.S. Auto Sales Data

## Course
CS 4412 — Data Mining

## Description
This project applies data mining techniques to discover temporal patterns, structural
groupings, and anomalous behavior in U.S. auto sales data. The focus is on pattern
discovery and interpretation using unsupervised methods rather than predictive
modeling.

## Dataset
Auto Sales — Bureau of Transportation Statistics  
https://data.transportation.gov/d/7n6a-n5tz (https://catalog-beta.data.gov/dataset/auto-sales)

## Techniques
- Clustering (K-Means, Hierarchical)
- Association Rule Mining (Apriori / FP-Growth)
- Anomaly Detection
- Dimensionality Reduction (PCA)
